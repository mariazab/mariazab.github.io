Trivia Quiz

Previous Next

Trivia Quiz is a game app where users play a quiz of 10 questions. The user can choose the category and the level. Based on the level and the correct answers, the score is calculated, which later can be saved as high score.

The app is built with React Native and Firebase is used for storing the high scores. The questions are retrieved from Open Trivia Database, which is a free database with trivia questions submitted by users.



Technologies: React Native, Expo, Firebase, React Navigation
