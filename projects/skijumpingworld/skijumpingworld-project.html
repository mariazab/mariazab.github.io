Ski Jumping World

Ski Jumping World is a project created for one of the courses at Haaga-Helia. It is a web service for ski jumping fans, which provides users with information on the latest results and world cup standings as well as athletes, hills and competitions. The user can also request an email with details on the upcoming competition. Additional feature is REST API.

The website is created with Java, Spring Boot, Thymeleaf, JawsDB and Bootstrap. It is hosted on Heroku and you may log in with the test user account (user/user).
